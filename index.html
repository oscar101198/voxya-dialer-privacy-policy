Politique de Confidentialité - Voxya Dialer

Dernière mise à jour : 2026-02-04

Introduction

Voxya Dialer ("nous", "notre", "l'application") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos données personnelles lorsque vous utilisez notre application mobile.

Informations que nous collectons

Données collectées automatiquement

L'application Voxya Dialer collecte et traite les données suivantes :

  • Contacts : Accès à votre carnet d'adresses pour permettre la numérotation et l'affichage des informations de contact lors des appels
  • Historique des appels : Accès à votre journal d'appels pour afficher l'historique et permettre la relance rapide
  • Données d'appel : Informations sur les appels entrants et sortants (numéros, dates, durées)

Données audio collectées

L'application utilise la permission RECORD_AUDIO pour les fonctionnalités suivantes :

  1. Enregistrement d'appels : L'application peut enregistrer vos conversations téléphoniques via la fonctionnalité de conférence à 3 de votre opérateur
  2. Mémos vocaux : L'application permet l'enregistrement de mémos vocaux locaux

Utilisation des données

Utilisation de la permission RECORD_AUDIO

La permission android.permission.RECORD_AUDIO est utilisée exclusivement pour :

  • Enregistrement d'appels : Capturer l'audio des conversations téléphoniques lorsque vous activez cette fonctionnalité
  • Mémos vocaux : Permettre l'enregistrement de notes vocales personnelles

Important : L'enregistrement d'appels n'est activé que lorsque vous l'activez manuellement dans l'application. L'application n'enregistre jamais automatiquement vos appels sans votre consentement explicite.

Stockage des données

  • Données audio : Les enregistrements d'appels et les mémos vocaux sont stockés uniquement sur votre appareil dans le stockage local de l'application
  • Contacts et historique : Ces données restent sur votre appareil et ne sont pas transmises à nos serveurs
  • Aucune transmission : Aucune donnée audio, contact ou historique n'est transmise à des serveurs externes ou à des tiers

Partage de données

Nous ne partageons, ne vendons ni ne louons vos données personnelles à des tiers.

  • Aucune donnée n'est transmise à des serveurs externes
  • Aucune donnée n'est utilisée à des fins publicitaires
  • Aucune donnée n'est partagée avec des services d'analyse tiers
  • Les enregistrements audio restent exclusivement sur votre appareil

Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Stockage local sécurisé sur votre appareil
  • Accès aux données protégé par les permissions du système Android
  • Aucune transmission de données sensibles via le réseau

Vos droits

Vous avez le contrôle total sur vos données :

  • Désactiver l'enregistrement : Vous pouvez désactiver l'enregistrement d'appels à tout moment dans les paramètres de l'application
  • Supprimer les enregistrements : Vous pouvez supprimer les enregistrements audio directement depuis l'application
  • Révoquer les permissions : Vous pouvez révoquer les permissions de l'application à tout moment via les paramètres Android
  • Désinstaller l'application : La désinstallation de l'application supprime toutes les données associées stockées localement

Permissions de l'application

L'application demande les permissions suivantes pour fonctionner :

  • READ_CONTACTS / WRITE_CONTACTS : Pour accéder et gérer votre carnet d'adresses
  • READ_PHONE_STATE : Pour détecter l'état des appels (en cours, terminé, etc.)
  • CALL_PHONE : Pour passer des appels téléphoniques
  • READ_CALL_LOG / WRITE_CALL_LOG : Pour afficher et gérer l'historique des appels
  • RECORD_AUDIO : Pour enregistrer des appels et créer des mémos vocaux (uniquement avec votre consentement)
  • ANSWER_PHONE_CALLS : Pour répondre aux appels entrants
  • POST_NOTIFICATIONS : Pour afficher les notifications d'appels

Toutes ces permissions sont nécessaires au fonctionnement de base d'une application de téléphonie.

Avertissement légal concernant l'enregistrement

Important : L'enregistrement de conversations téléphoniques peut être soumis à des réglementations légales selon votre juridiction. Il est de votre responsabilité de :

  • Informer vos correspondants que vous enregistrez la conversation si la loi l'exige dans votre pays
  • Respecter les lois locales concernant l'enregistrement d'appels
  • Obtenir le consentement de toutes les parties avant d'enregistrer une conversation lorsque requis par la loi

Nous ne sommes pas responsables de l'utilisation non conforme de la fonctionnalité d'enregistrement.
